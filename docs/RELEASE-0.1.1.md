# DesktopCal Lightweight 0.1.1

## 版本定位
`0.1.1` 是在 `0.1.0` 首个可打包基线版本之上的一次体验修订版。

这一版不扩张产品边界，重点放在桌面挂件体验的收口：
- 收起为右上角紧凑日期条
- 记住收起 / 展开状态
- 改善紧凑条布局与动画
- 修复导航时滚动条闪现带来的窗口抖动

## 主要更新
### 新增
- 收起态：完整日历可收起为右上角紧凑条
- 收起态内容：仅显示 `几月几日 + ∨`
- 展开态内容：保留完整五周日历
- 收起状态持久化：应用重启后保持上次收起 / 展开状态
- 展开尺寸持久化：收起后再展开会恢复上次完整窗口尺寸

### 优化
- 收起 / 展开动画改为“旧内容先淡出，新内容后淡入”
- 窗口几何动画步数提升，右上锚定过渡更平滑
- 收起条最小宽度提高到 `320px`
- 收起条按钮布局更稳定，避免展开按钮被挤出可视范围
- 禁用全局滚动条显示，切换周视图时不再闪出滚动条并引起抖动

## 当前能力
- 五周滚动视图：当前周前 1 周、后 3 周
- 双击日期格进入原位扩展编辑态
- 紧凑任务输入与完成勾选
- 单日事项 SQLite 本地持久化
- 日期摘要与超出计数
- 托盘显示、隐藏、设置、退出
- 关闭主窗口自动隐藏到托盘
- 无边框、右上锚定、跳过任务栏
- 可配置窗口尺寸与右上偏移
- 开机自启动开关
- 单实例运行，重复启动时自动唤醒现有窗口
- 自定义应用图标与托盘图标

## 当前交互模型
- 单击日期：选中
- 双击日期：进入编辑态
- 编辑态点击空白：自动保存并关闭
- `↑`：向前 2 周
- `↓`：向后 2 周
- `⌂`：回到今天所在窗口
- `⚙`：打开设置
- `∧`：收起为紧凑条
- `∨`：展开为完整日历

## 默认参数
- 默认窗口大小：`1300 x 850`
- 收起态最小宽度：`320px`
- 默认锚点：主显示器工作区右上角
- 默认偏移：`右 5px / 上 5px`
- 默认运行形态：无边框、透明窗口、托盘常驻

## 存储说明
- 日程数据库：SQLite
- 设置存储：本地 JSON
- Windows 默认落点：
- `%APPDATA%\\com.lcy.desktopcal.lightweight\\settings.json`
- `%LOCALAPPDATA%\\com.lcy.desktopcal.lightweight\\desktopcal.sqlite3`

## 打包说明
推荐命令：

```bash
pnpm tauri build
```

默认构建产物目录：
- `src-tauri/target/release/bundle/`

## 已知限制
- 当前为 Windows 优先实现
- 节假日仍是内置静态数据
- 当前仍是“桌面挂件式”常驻，不是真正附着到 WorkerW / 桌面图标层之后
- 项目中仍有少量历史编码残留文案，后续应继续统一清理
